Wedding Invitation Brand Identity: 'Squirrel' Theme1. КонцепцияСайт-приглашение должен ощущаться как дорогой бумажный носитель с сургучной печатью, перенесенный в веб. Минимализм здесь — не отсутствие деталей, а их выверенная точность. Используем «грязноватые», глубокие землистые тона палитры Squirrel для создания атмосферы уюта и фундаментальности.2. ТипографикаЗаголовки: Классический высококонтрастный Serif (например, Playfair Display или Cormorant Garamond). Тонкий, элегантный, с широким межстрочным интервалом.Основной текст: Гротеск с гуманистическими чертами (например, Inter или Montserrat), вес 300 или 400. Отступ между буквами (letter-spacing) +0.05em.4. Визуальные элементыСургучная печать: Использовать как интерактивный элемент (например, кнопка «Открыть приглашение» или маркер завершения блока). Анимация должна имитировать мягкое нажатие.Разделители: Тонкие линии толщиной 0.5px — 1px в цвете #dad3cd (300).Фото: Предпочтительно черно-белое или с десатурированными цветами, чтобы не конфликтовать с теплой палитрой Squirrel.Промпт для генерации дизайна (Midjourney / DALL-E)Скопируй этот промт, чтобы получить визуальный референс интерфейса:MarkdownUI/UX design for a high-end wedding invitation website, "Squirrel" color palette, dominant soft beige #fbfbf8 background, deep brown #201913 text, elegant serif typography, clean minimalist layout, luxury aesthetic, interactive wax seal element, plenty of negative space, professional web design, 8k, high fidelity, sophisticated and timeless vibe --ar 9:16 --v 6.0
